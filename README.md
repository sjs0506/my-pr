추가로 개발 경험을 통해 아래 역량을 보유하고 있습니다.

| 분야 | 내용 |
|------|------|
| API 설계 | REST API 설계 및 엔드포인트 구조 구성 경험 |
| 통신 | HTTP / WebSocket 통신 기본 개념 및 적용 경험 |
